# TourSystem

## 基于协同过滤的旅游推荐系统

架构：Django、Bootstrap、mysql

### 功能描述：
* 用户登录注册
* 管理员管理后台数据(django自带)
* 爬虫爬取"去哪儿网"景点数据并存储
* 爬虫爬取"去哪儿网"旅游攻略帖子
* 景点及详情展示
* 根据用户历史点击记录进行基于物品协同过滤推荐
